Viewing Group Inventory

Available in Open Source.

This inventory screen allows users to view and manage a selected group. The screen displays a group's user-specified properties, the resources contained within the group, and the roles assigned to the group. The group's name is at the rightmost of the path listed at the top of the screen.

Tasks Available on This Screen

On this screen, users can do the following tasks:

To view a map representation of the group's resources:

  • Click Map in the upper left.
    From there you can navigate to any one of the group's resources by clicking on the resource's name.

To change the group's owner:

  • In the middle near the top of the screen, click Change next to the listed owner and proceed to the "Edit" screen.

To change the group's general properties:

  • In the "General Properties" section, click Edit and proceed to the "Edit" screen.

To add a resource to the group:

  • In the "Resources" section, click Add to List and proceed to the "Add to Groups" screen.

To remove a resource from the group:

  • In the "Resources" section, check the resource and click Remove from List.

To assign a role to the group:

  • In the "Roles Assign To" section, click Add to List and proceed to the "Add Roles to Groups" screen.

To remove a role from the group:

  • In the "Roles Assigned To" section, check the role and click Remove from List.

To add a new group:

  • In the Tools Menu in the upper right, select New Group.

To add this group to the user's Dashboard favorites:

  • In the Tools Menu in the upper right, select Add to Dashboard Favorites.

Sections on This Screen

General Properties

This section lists the properties the user specified when creating or last modifying the group:

  • Owner: Defaults to the account that created the group
  • Description: A description of the group
  • Location: The physical location of the group's hardware
  • Resource type: The type of resource that this group comprises. It can be a single resource type, for example, Exchange 3000, or a mix of types, for example, servers, services, and applications.

Resources

This section lists the group's resources. To view a resource listed here, click the resource's name. The resource's availability is also indicated.

Roles Assigned To

This section lists the roles assigned to the group. Only users assigned to the selected role(s) will be able to monitor this group. To view a role listed here, click the role's name.

"Control" configuration properties are data that HQ must have in order for users to perform available control actions on this group. Only compatible groups that contain a resource type that itself has associated control actions will have any control configuration properties specified here.

HQ often provides reasonable defaults for the configuration properties, but HQ users must provide information specific to their environment.

Navigating via the Top-Level Menu

Main screens in Hyperic contain the same menu at the top of the page. Users can navigate to the following parts of HQ using this menu:

Dashboard Takes users to the starting screen in HQ, which displays information about resource health, recent alerts, recently performed Auto-Discoveries, and control actions.
Browse Resources Takes users to the screen where they can locate and navigate through information about all managed resources
Administration Takes users (who are assigned to a role that has the Administer Hyperic HQ Server Configuration option enabled) to the screen where they can manage the Hyperic HQ Server configuration.
Recently Viewed Resources Provides a drop-down list of resources that the user has recently viewed. Selecting one of those resources takes the user to the inventory of that resource.
Help Opens screen-sensitive online help in a secondary window
