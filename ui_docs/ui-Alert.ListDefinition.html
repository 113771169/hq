List Alert Definitions

This page displays the alerts that have been configured for the specific resource or resource type.

From this page you can create new alert definitions by clicking the button. This page allows you to quickly activate or deactivate multiple alert definitions simultaneously by checking the checkbox next the alert(s), selecting YES or NO in the dropdown box, and clicking . Alert definitions that are no longer needed can be deleted by checking the appropriate checkboxes and clicking .

If you are viewing a list of alert definitions for a specific resource, clicking the button will take you to the HQ List Alerts page.
