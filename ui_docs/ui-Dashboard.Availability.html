HQ Dashboard Settings: Availability Summary

Available through Open Source

This screen allows users to control which and how resources are displayed in the Dashboard's "Availability Summary" portlet.

Tasks Available on This Screen

On this screen, users can do the following tasks:

To specify a header description to appear in the portlet:

  • Type a Description and click OK.

To change the number of resource types displayed in the portlet:

  • Select a Display Range value and click OK.

To add a resource to the portlet:

  • In the "Selected Resources" section, click Add to List and proceed to the "Add/Remove Resources" screen.

To remove a resource from the portlet:

  • In the "Selected Resources" section, check the resource and click Remove from List.
