HQ Dashboard Settings: Recently Added

Available through Open Source

This screen allows users to change the Dashboard display of platforms added to the HQ inventory in the last two days.

Tasks Available on This Screen

On this screen users can do one thing: change the number of platforms listed in the Dashboard's "Recently Added" portlet.

To change the number of platforms displayed:

  • Select the number of recently added platforms to display and click OK.
