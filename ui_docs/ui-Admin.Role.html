Viewing and Editing a Role

Available in Enterprise Edition.

This screen allows users to view and manage an individual role. This screen displays the permissions set for the role, the groups of resources assigned to the role, and the users assigned to the role.

  • Tasks available on this screen
  • Sections on this screen
  • Navigating via the top-level menu

Tasks Available on This Screen

On this screen, users can do (or at least start) the following tasks:

To change the role's properties or permissions:

  • In the "Properties & Permissions" section, click Edit and proceed to the "Edit" screen.

To assign a user to the role:

  • In the "Assigned Users" section, click Add to List and proceed to the "Add Users to Roles" screen.

To remove a user from the role:

  • In the "Assigned Users" section, check the user and click Remove from List.

To assign a group to the role:

  • In the "Assigned Groups" section, click Add to List and proceed to the "Add Groups to Roles" screen.

To remove a group from the role:

  • In the "Assigned Groups" section, check the group and click Remove from List.

Sections on This Screen

Properties & Permissions

This section lists basic information about the role:

  • Name
  • Owner
  • Description
  • Administer HQ Server Configuration: Indicates whether the users in this role may configure the HQ Server Settings. This is NO by default.

Assigned Users

This section lists the users assigned to this role. To view a user listed here, click the user's name. Users can add and remove users from the role.

Assigned Groups

This section lists the groups assigned to this role. To view a group listed here, click the group's name. Users can add and remove groups from the role.

The Final Step

Even if users are assigned to the role, the role's (and users') permissions will not apply to any specific resources until a group (of said specific resources) is also assigned to the role.

Navigating via the Top-Level Menu

Main screens in Hyperic contain the same menu at the top of the page. Users can navigate to the following parts of HQ using this menu:

Dashboard Takes users to the starting screen in HQ, which displays information about resource health, recent alerts, recently performed Auto-Discoveries, and control actions.
Browse Resources Takes users to the screen where they can locate and navigate through information about all managed resources
Administration Takes users (who are assigned to a role that has the Administer Hyperic HQ Server Configuration option enabled) to the screen where they can manage the Hyperic HQ Server configuration.
Recently Viewed Resources Provides a drop-down list of resources that the user has recently viewed. Selecting one of those resources takes the user to the inventory of that resource.
Help Opens screen-sensitive online help in a secondary window
