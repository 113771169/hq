View Control-Action History

Available in Open Source.

This screen allows users to view the history of control actions performed on a resource.

Tasks Available on This Screen

On this screen, users can do the following tasks:

To delete the record of a control action already performed on a resource:

  • Check the Control Action and click Delete.

To configure the server for monitoring and control:

  • In the Tools Menu, click Configure Server and proceed to the "Edit Resource Configuration" screen.

To add this server to the Dashboard Favorites:

  • In the Tools Menu, click Add to Dashboard Favorites.

To create a new service that will be added to this server:

  • In the Tools Menu, click New Service and proceed to the "Creating a New Service" screen.
