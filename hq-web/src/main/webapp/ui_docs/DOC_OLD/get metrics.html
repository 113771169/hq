Overview

The getMetric API returns min/max/avg values for one or more metrics over a specified timeframe. The data is extracted from the raw data and aggregated data DB tables.

Data os extracted from the databsae table that contains the aggregated from the lowest interval / raw data (the later represents the lowest interval), in which no more than 400 data points fits in the requested time frame.

API Details

Object Metric
URL metric
Method post
Parameters measurementsRequest
  • start time
  • end time
  • a resource id
  • list of measurement template names
Returned Object measurementResponse
  • error-code (if resource fails)
  • description (if resource fails)
  • ID
  • list of Measurements (no measurements will be calculated for measurementTemplates that are unknown)
