This page last changed on Jan 04, 2012 by mmcgarry.

Topics marked with * relate to features available only in vFabric Hyperic.

To edit a resource's name or description.

  1. Navigate to the resource's Inventory page,
  2. Click Edit the General Properties section of the page.
  3. Make desired edits on the General Properties page.
  4. Click OK.

The resource owner is shown on the General Properties page. An authorized user can change the owner of a resource on its Inventory page. If a user has the right to modify a resource's owner, a Change link is available next to the Owner field in the header of the Inventory page,
