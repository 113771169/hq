vFabric Hyperic Product Plug-in Development documents the XML plug-in descriptor that is the basis of every plug-in, Hyperic support classes for auto-discovery, measurement, control, and other management functions. It provides information about developing VMware vFabric™ Hyperic® and Hyperic HQ product plug-ins to manage new resource types.

Intended Audience

This guide is intended for developers who build or customize Hyperic plug-ins.
