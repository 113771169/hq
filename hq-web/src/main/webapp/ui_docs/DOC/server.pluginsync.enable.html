Hyperic 4.6 Documentation : server.pluginsync.enable
This page last changed on Apr 28, 2011 by mmcgarry.

This page describes a feature available in the beta release of Hyperic 4.6, and is subject to update and correction.

Description
The server.pluginsync.enable property can be used to disable Hyperic's Server-Agent Plugin Synchronization (SAPS) feature, described in Plugin Deployment and Management in Hyperic 4.6.

To disable SAPS, add this property setting to ServerHome/conf/hq-server.conf, and restart the Hyperic Server:

server.pluginsync.enable

Default

SAPS is enabled by default. Note however that hq-server.conf does not contain the server.pluginsync.enable property. You must add this property to the configuration file to disable SAPS.
