This page describes pages in the VMware vFabric™ Hyperic® user interface that are specific to particular resource types — these are views enabled by the product plugin that manage the resource type, and are present in the Hyperic user interface only if you have resources of that type under management.

  • HQ vSphere View
  • tc Server Application Management View
