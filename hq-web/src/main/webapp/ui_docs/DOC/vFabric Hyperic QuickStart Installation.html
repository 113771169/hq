vFabric Hyperic 4.6 : vFabric Hyperic QuickStart Installation
This page last changed on Jun 30, 2011 by mmcgarry.
Note: If you obtain vFabric Hyperic as part of a vFabric Cloud Application Platform package, first complete the license activation and installation procedures in Getting Started with vFabric Cloud Application Platform. Then follow procedures in this document to set up your environment for vFabric Hyperic and complete any remaining Hyperic-specific installation and configuration tasks.

To evaluate Hyperic components, you can get up and running quickly by installing the Hyperic Server and a Hyperic Agent on the same host, using Hyperic's built-in database.

For comprehensive installation information, see vFabric Hyperic Installation Guide.

Procedure

  1. Verify Installation Requirements for the Hyperic Server and Hyperic Agent.
  2. Choose and Download an Installation Package. Choose a "full" installer.
  3. Install the components as described in Installing the Agent and Server from a Tarball or Zip Archive.
  4. For post-installation tips, review What to Do After Installing the HQ Server and HQ Agent.
  5. Start the Hyperic Server: Starting the Server on Unix-Based Platforms or Starting the Server on Windows To Run as a Service.
  6. Open the Hyperic portal at 
    http://localhost:7080/
  7. Log on, using the Hyperic username and password specified during Hyperic Server installation.
  8. Start and configure the Hyperic Agent as described in Configure Agent - Server Communication Interactively.
