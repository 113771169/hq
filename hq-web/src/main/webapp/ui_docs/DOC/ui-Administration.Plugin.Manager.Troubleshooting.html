Troubleshooting

The Out-of-date installed Agents popup lists the name and version of each agent that is reporting to the Hyperic Server but not synchronizing plugins. The Hyperic Agent must be the same or later version as the Hyperic Server for plugin synchronization to occur. For more information, see Plugin Deployment and Management.
