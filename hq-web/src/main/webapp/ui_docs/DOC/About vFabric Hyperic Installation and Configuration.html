vFabric Hyperic Installation and Configuration provides procedures for installing, configuring, and managing the Hyperic Server, Agent, and database; configuring global alert and metric processing options; and scaling and supporting the Hyperic infrastructure.

Intended Audience

vFabric Hyperic Installation and Configuration is intended for operations personnel who set up and support the core Hyperic infrastructure.
