Hyperic 4.6 Documentation : ui-Admin.User.Password
This page last changed on Dec 15, 2010 by mmcgarry.

The Change Password page allows you to change a user's account password.

To change a password:

  1. If you are do not have the SuperUser role, the page prompts you to supply the previous password.
  2. Enter a password in the New Password field.
  3. Enter a password again in the Confirm New Password field.
  4. Click OK.
