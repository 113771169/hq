About the Recently Added Portlet

The Recently Added portlet lists platforms that have recently been added to inventory.

Configure the Recently Added Portlet

  1. Click in the upper right corner of the Recently Added portlet.
  2. The HQ Dashboard Settings: Recently Added page appears.
  3. In the Show Maximum of: pulldown, select the maximum number of platforms to list in the portlet.
  4. Click OK.
