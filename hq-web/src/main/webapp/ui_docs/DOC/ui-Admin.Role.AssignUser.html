Assigning Users to Roles

Available only in HQ Enterprise

This screen allows Administrators to assign users to an existing role. For users to have access to anything in the system, they must be assigned to a role, from which they get all their access rights.

Feedback is welcome. Click Add Comment at the bottom of the page.

Tasks Available on This Screen

On this screen, users can do one thing: assign users to a role.

To assign a user to a role:

  1. Check the desired users in "Users" and click between the two lists to move the selected roles to "Assign To Role." (The arrow is enabled only when some roles are selected.)
