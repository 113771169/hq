Changing a Resource's Owner

Topics marked with * relate to HQ Enterprise-only features.

This screen allows users to change the owner of a single resource. In order to change a resource's owner, you must either have Administrator access or be the resource owner. (There is no facility for changing multiple resource's owners in one shot.)

Feedback is welcome. Click Add Comment at the bottom of the page.

Tasks Available on This Screen

On this screen, users can do one thing: change a resource's owner.

To change a resource's owner:

  • Click next to the new owner's name.
