vFabric Hyperic 4.6.5 : Resources, Resource Types and Inventory Types
This page last changed on Jan 04, 2012 by mmcgarry.

Topics marked with * relate to features available only in vFabric Hyperic.

An individual managed resource is classified in Hyperic inventory as an inventory type and and as a resource type. An inventory type relates to a software dependency hierarchy, most notably, Hyperics's platform - server - service hierarchy. Groups and applications are examples of inventory types. A resource type relates to the "brand" or vendor associated with a resource.
