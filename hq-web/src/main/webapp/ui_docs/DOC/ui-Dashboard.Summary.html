Topics marked with * relate to HQ Enterprise-only features.

Pre-Release Documentation
This space is a staging area for Hyperic 4.5 documentation. See HQ Documentation for Hyperic 4.2, 4.3, and 4.4 documentation.

This screen allows users to change which resources are counted in the "Summary Counts" portlet on the Dashboard.

Feedback is welcome. Click Add Comment at the bottom of the page.

Tasks Available on This Screen

On this screen users can do one thing: change which resources are counted in the portlet.

To change the resource count:

  1. For each inventory level, check either:
    • "Show Total of All": All resources at that inventory level will be included in the count
    • A subset of all the listed resource types for that inventory level: Only resources of the selected types will be included in the count
  2. Click .
Return to the Dashboard.
