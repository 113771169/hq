Views Tab for a Resource

The Views tab for a resource is the attach point for HQU plugins for a resource type; it has a button for each HQU plugin attached to the current resource type. For example, when an platform is selected, the Views tab has a Live Exec button for querying the platform in real-time.

If there are no HQU plugins available for the type of resource currently selected, the Views tab will not have any buttons.

HQU plugins are Groovy-based HQ extensions that can be attached:

  • to the Views tab for a resource type,
  • as an item on a Masthead tab menu, or
  • in the "Plugins" section of HQ's Administration tab.

HQ has a number of built-in HQU plugins. Users can also develop and deploy custom HQU plugins.

For more information about HQU plugins see vFabric Hyperic HQU Plug-in Development.
