This section lists the properties for configuring vFabric Hyperic for an SNMP server, which can be used for alert notification once SNMP traps are enabled, as described on Enable vFabric Hyperic to Send SNMP Traps. The display of configuration properties depends on the selected SNMP Protocol Version (if any).

For version 3:

  • Trap OID
  • Auth Protocol
  • Auth Passphrase
  • Privacy Protocol
  • Privacy Passphrase
  • Target Context Name
  • Security Name
  • Engine Id

For version 2c:

  • Trap OID
  • Community

For version 1:

  • Trap OID
  • Community
  • Generic ID
  • Specific ID
  • Enterprise ID
  • Agent Address
