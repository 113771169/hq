Assigning Roles to Users

Available only in HQ Enterprise

Pre-Release Documentation
This space is a staging area for Hyperic 4.5 documentation. See HQ Documentation for Hyperic 4.2, 4.3, and 4.4 documentation.

This screen allows Administrators to assign existing roles to an existing user. For users to have access to anything in the system, they must have a role, from which they get all their access rights.

Feedback is welcome. Click Add Comment at the bottom of the page.

Tasks Available on This Screen

On this screen, users can do one thing: assign roles to a user.

To assign a role to a user:

  1. Check the desired roles in "Roles" and click between the two lists to move the selected roles to "Add To Roles." (The arrow is enabled only when some roles are selected.)
