vFabric Hyperic 4.6 : ui-Admin.User.AssignRole
This page last changed on Jun 07, 2011 by mmcgarry.

To assign roles to a user:

  1. In the "Roles" panel on the left side of the page, checkmark each role to which you want to assign the user, and click the blue arrow to move the roles to the "Add to Roles" panel.
  2. Click OK when you are done adding users to the role.
