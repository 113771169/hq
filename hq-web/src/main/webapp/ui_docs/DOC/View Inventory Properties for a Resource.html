vFabric Hyperic 4.6.5 : View Inventory Properties for a Resource
This page last changed on Feb 22, 2012 by mmcgarry.

In the Hyperic user interface, a resource's Inventory Page presents a variety of resource properties and key information about the resource's closest "relatives" in the resource hierarchy. The information on the Inventory page varies by both inventory type and resource type, and includse auto-discovered and user-configured resource properties. For a fundamental inventory type — a platform, serve, or service — the Inventory Page contains a "Configuration Properties" section, where you can view and update the properties that enable or tailor monitoring behaviors for the resource.

See the pages below for information about the Inventory page for each inventory type.
