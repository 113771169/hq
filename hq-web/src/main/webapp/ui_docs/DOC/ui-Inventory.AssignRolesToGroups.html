Assign Roles to a Group


To add a role to a group:

  1. Navigate to the group's Inventory page, if it is not currently selected.
  2. Click Add To List in the Roles section of the page.
  3. On the Add to Roles to Group page, checkmark desired roles and click the blue arrow to move them from the Roles column to the Add To Roles column. (The arrow is enabled when you select a role.)
  4. After moving desired roles to the Add Resources column, click OK.
