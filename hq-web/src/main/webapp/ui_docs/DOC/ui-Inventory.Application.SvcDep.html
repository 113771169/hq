Configuring Application-Service Dependencies

Topics marked with * relate to HQ Enterprise-only features.

This screen allows users to view and select the services that the application's service depends on and to view, in turn, the services that depend on this application's service.

  • Tasks available on this screen
  • Sections on this screen

Feedback is welcome. Click Add Comment at the bottom of the page.

Tasks Available on This Screen

On this screen, users can do the following tasks:

To add a service that this service depends on:

  • In the top section, click and proceed to the "Edit Application" screen.

To remove a service from the list of services this service depends on:

  • In the top section, check the service and click .

To view the dependencies of a service that is listed as a dependency of the current service:

  • Click next to the service.

Sections on This Screen

<Service Name> Depends on

This section lists all the other services that this particular service depends on.

  • Dependencies: This button returns the user to this screen, but with information for the selected service
  • Service: The service name
  • Service Type: The service type
  • Host Server: The server that hosts this service

Services That Depend on <Service Name>

This section lists all the services that depend on this particular service. All the service identifying information is the same as for the previous section.
