Adding Resources to a Group

Topics marked with * relate to HQ Enterprise-only features.

Pre-Release Documentation
This space is a staging area for Hyperic 4.5 documentation. See HQ Documentation for Hyperic 4.2, 4.3, and 4.4 documentation.

This screen allows users to add resources to the current group.

Which Resources Can Be Added? The only resources available for addition to the group are those of the same resource type specified for this group.

Feedback is welcome. Click Add Comment at the bottom of the page.

Tasks Available on This Screen

On this screen, users can do one thing: add resources to a group.

To add resources to a group:

  1. Check the desired resources in "Resources" and click between the two lists to move the selected resources to "Add Resources." (The arrow is enabled only when some resources are selected.)
    Users can filter the list of all available resources either by:
    • Typing in (part of) the resource's name in Filter By Name and clicking directly to the right, or
    • Selecting a resource or inventory level in Filter By Type. (This filter is available only for mixed groups, as compatible groups contain only one resource type.)
