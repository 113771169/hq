Editing an Application's Contact Information

Topics marked with * relate to HQ Enterprise-only features.

This screen allows users to edit an application's contact information and business owner. IT administrators should assign responsibility for this application according to their own IT and business practices.

Feedback is welcome. Click Add Comment at the bottom of the page.

Tasks Available on This Screen

On this screen, users can do one thing: edit contact and responsibility information for the application.
