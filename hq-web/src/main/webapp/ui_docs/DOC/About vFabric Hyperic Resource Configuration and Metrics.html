vFabric Hyperic Resource Configuration and Metrics documents the configuration requirements and metrics available for selected VMware vFabric™ Hyperic® and Hyperic HQ product plug-ins. This guide supplements vFabric Hyperic Administration.

Intended Audience

vFabric Hyperic Resource Configuration and Metrics is intended for Hyperic administrators who configure resources for monitoring.
