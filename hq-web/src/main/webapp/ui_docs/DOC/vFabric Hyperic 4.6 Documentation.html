The vFabric Hyperic 4.6 documentation contains these topic sets:

  • vFabric Hyperic 4.6 Beta Release — Has links to new and changed content related to Hyperic 4.6 features.
  • vFabric Hyperic 4.6 Release Notes — Release Notes for Hyperic 4.6.
  • vFabric Hyperic Overview — This is a guide to Hyperic functionality.
  • vFabric Hyperic Installation and Configuration — For operations personnel that set up and support the core Hyperic infrastructure. This is a guide to installing, configuring, and managing the Hyperic server, agent, and database; configuring global alert and metric processing options; and scaling and supporting the Hyperic infrastructure.
  • vFabric Hyperic Administration — For Hyperic administrators who implement and support the Hyperic monitoring envionment. This is a guide to getting resources into inventory, configuring monitoring and alerting, and managing users, and in vFabric Hyperic, roles.
  • vFabric Hyperic Resource Configuration and Metrics — Plug-in-specific reference material that supplements Fabric Hyperic Administration. For selected product plug-ins, this topic set documents the configuration (if any) required to manage resources, and the metrics that Hyperic can collect for a each resource type the plug-in manages.
  • vFabric Hyperic Monitoring and Management — For operations personnel that monitor resources with Hyperic. It has information about using the Hyperic monitoring user interface, performing resource control actions, and processing fired alerts.
  • vFabric Hyperic User Interface — For users that interact with the Hyperic user interface. This topic set contains the on-line help for each page in the Hyperic user interface.
  • vFabric Hyperic Web Services API — For administrators or support personnel that automate Hyperic configuration and administration processes, or integrate Hyperic with other systems. It provides instructions for running each API from the command-line and introductory information for calling the APIs from Java.
  • vFabric Hyperic Product Plug-in Development — For developers that customize Hyperic plugins or build new plugins. It documents the XML plugin descriptor that is the basis of every plugin, and has information about Hyperic support classes for auto-discovery, measurement, control, and other management functions.
  • vFabric Hyperic HQU Plug-in Development — For Groovy developers that use the HQU framework to extend the Hyperic user interface or Hyperic management capabilities.
