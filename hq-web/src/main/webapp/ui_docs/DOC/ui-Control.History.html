View Control-Action History

Topics marked with * relate to HQ Enterprise-only features.

This screen allows users to view the history of control actions performed on a resource. The screen displays the following information for each performed control action:

  • Control Action: The action performed on the resource
  • Arguments: Arguments used (if any) in performing the control action
  • Command State: Status of the action. Possible values are: Completed, In Progress.
  • Date Started: The date and time the control action was initiated
  • Elapsed Time: How long it took to perform the control action
  • User: Username of the user who initiated (or scheduled) the control action
  • Message (optional): Additional information about the control action

Feedback is welcome. Click Add Comment at the bottom of the page.
