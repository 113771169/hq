This screen allows users to edit the general properties of an individual user and to enable the user account for login.

Feedback is welcome. Click Add Comment at the bottom of the page.

Tasks Available on This Screen

On this screen, users can do the following tasks:

To edit a user's general properties:

  • Change any of the following values and click :
    • Name
    • Email
    • Phone
    • Department
    • SMS Address
