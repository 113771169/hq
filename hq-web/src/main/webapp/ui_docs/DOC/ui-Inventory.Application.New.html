Create an Application

  1. Click Browse on the Resources tab.
  2. Click New Application in the Tools menu.
    • The New Application page appears.
  3. In the "General Properties" section of the *New Application page, enter:
    • Name — Supply the name of the application.
    • Description — Enter a description of the application, if desired.
    • Location — Enter the location of the application, if desired.
  4. In the "Application Properties" section of the New Application page, provide desired contact information:
    • Engineering Contact
    • Business Owner
    • IT Contact
  5. Click OK.
    • The Inventory page for the application appears
