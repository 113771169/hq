Configuration Tracking Plugin

Topics marked with * relate to features available only in vFabric Hyperic.

The Configuration Tracking Plugin implements tracking of configuration files.

An example Plugin using Configuration Tracking Plugin is xinetd Plugin:

xinetd-plugin.xml
... 
<plugin type="config_track"
            class="org.hyperic.hq.product.ConfigFileTrackPlugin"/>
 ...

Properties

Property Use
DEFAULT_CONFIG_FILE default config file to track  
DEFAULT_CONFIG_TRACK_ENABLE config tracking enabled by default
