Tasks Available on This Screen

On this screen, users can do one thing: add a service to an application.

To add a service to an application:

  1. Check the desired services in "Services" (all managed services are listed here) and click between the two lists to move the selected resources to "Add Services." (The arrow is enabled only when some resources are selected.)
    Users can filter the list of all available resources either by typing in (part of) the service's name in Filter By Name and clicking directly to the right.
