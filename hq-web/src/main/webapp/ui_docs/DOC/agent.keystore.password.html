vFabric Hyperic 4.6 : agent.keystore.password

vFabric Hyperic 4.6 : agent.keystore.password
This page last changed on Jul 19, 2011 by mmcgarry.

Description

This property configures the password for a Hyperic Agent's SSL keystore. Define the location of the keystore using the agent.keystore.path property.

These values of agent.keystore.path and agent.keystore.password can only be supplied by defining them in agent.properties.

Best Practice for Hyperic Agent Keystore
Before starting a Hyperic 4.6 Agent for the first time:
  • Create a keystore on its host with an SSL certificate signed by the same CA as the Hyperic Server's SSL certificate.
  • Configure the location and password for the agent keystore using agent.keystore.password and agent.keystore.password.

For more information see Hyperic Security Features and Recommendations.
