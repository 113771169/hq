Alert Detail Page

To view the Alert Detail page for a fired alert, you can:

  • click the date link for an alert in the Recent Alerts dashboard portlet,
  • click the date link for an alert in a resource's Alert tab, or 
  • click the alert definition name for an alert in the Alert Center.

The Alert Detail page summarizes  key information about a fired alert - why it fired, what automatic actions and notifications were performed, responder comments, and current status.  As appropriate, you can mark an alert "Fixed" and acknowledge an alert with an escalation.

Understanding Alert Details

  • Alert Properties - This section tells you the alert definition that fired the alert, the affected resource, the alert's priority and whether it has been fixed. You can click the alert definition name to view it, or the resource name to view metrics and charts.
  • Condition Set - This section shows the condition and firing rules from the alert definition, and the metric value ("Actual Value") or event that made the alert fire. 
  • Control Action - If HQ performed an automatic control action when the alert fired (as may be be specified in an alert definition) the action is listed here. 
  • Notification Actions - This section lists any notification actions that HQ performed for the fired alert, for example emails to HQ users. 
  • Escalation - If the alert has an escalation, this section lists the actions defined for the escalation, the steps that have been performed. Acknowledgment notes are shown, if available.
  • Fix - If the alert is fixed, the user that fixed it is listed. In HQ Enterprise, if the alert was fixed by a recovery alert, the Fix section will show "Recovery Alert". If the alert is unfixed, the Fixed button is available.

Mark Alert Fixed

In the "Fix" section, enter comments on the alert resolution, and click Fixed at the bottom of the page. The icon appears only if the alert is unfixed.

Acknowledge Alert

In the "Escalation" section, click the Acknowledge icon. The icon appears only if the alert has an escalation, and hasn't already been acknowledged.

Go to the Alert Definition Page

To view or edit the alert definition, click the alert definition name in the "Alert Properties" section of the page.

View Metrics and Other Alerts for the Resource

To view the Indicators page for the resource for which the alert fired, click the resource name in the "Alert Properties" section of the page. On the Indicators page, you can click the Alert tab to view a list of all fired alerts for the resource.
