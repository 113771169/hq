Configuration help:

Configure PostgreSQL 9.x for Monitoring

Monitoring of PostgreSQL is done through the statistics collector. For PostgreSQL to collect these metrics, make sure the following parameter is set to its default value:

track_counts = on

Make sure to restart PostgreSQL after making any changes to postgresql.conf.

Also note that some metrics like 'Index Space Used' and 'Database Space Used' will only be incremented after vacuum or analyze is run.

For more information see the PostgreSQL Documentation
