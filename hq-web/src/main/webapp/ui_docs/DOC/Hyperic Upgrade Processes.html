vFabric Hyperic 5.0 : Hyperic Upgrade Processes
This page last changed on Oct 02, 2012 by mmcgarry.

About this page...
This page is a guide to Hyperic upgrade procedures, which vary depending on the version and package of Hyperic you use.

For information about performing a new installation of Hyperic components, see Hyperic Installation and Startup Process.

  • Migrate Hyperic Server and Database to Version 5.0 — Follow the instructions on Migrate v4 Hyperic Server and Database to v5 to upgrade a Hyperic 4.x Server and database to version 5.0.
  • Upgrade from Hyperic 5.0 — Follow the instructions on Upgrade a Hyperic 5.0 Installation to upgrade a version 5.x Server to a later 5.x version.
  • Upgrade Hyperic vApp — To upgrade the Hyperic Server vApp, follow the instructions in the Upgrade Hyperic Server vApp section of the Hyperic Server vApp Administration Console page.
  • Upgrade Hyperic Agents — Follow the instructions on Upgrade Hyperic Agent.
