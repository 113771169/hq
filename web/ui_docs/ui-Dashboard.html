HQ Dashboard

Available in Open Source unless marked by * (available in Enterprise Edition)

The Hyperic HQ Dashboard is your starting point when you log in to Hyperic HQ. By default, it displays information about resource health, recent alerts, recently performed Auto-Discoveries, and control actions. You can also search for specific resources from the Dashboard, and display summary counts for all resources that Hyperic HQ is managing.

You can customize the Dashboard so that it displays only the information that matters to you when you first log in. Portlets can be removed from the Dashboard by clicking the icon on each portlet. Once a portlet is removed, it can be re-added from the Add content to this column: section at the bottom of the page. Portlets may also have configuration information which can be accessed using the icon on the top of each portlet. You can also re-arrange the portlets that are displayed by simply drag-and-dropping the header of each portlet, so that the information you are most concerned with will show up near the top of the page.

There are also a number of dashboard portlets that are capable of updating you through RSS. Those portlets that support notifications through RSS display the icon. Wherever you see this icon, you can click on or copy the link of the icon for the RSS feed URL.

HQ has the following portlets on the Dashboard:

Saved Charts

This portlet will display links to saved charts. Charts can be saved to the Dashboard from the HQ Chart page. To manage the current list of charts, click and proceed to the "HQ Dashboard Settings: Charts" screen.

Availability Summary

The availability summary portlet(s) will show the aggregate availability of selected resources categorized by resource type. To change the selected resources for the portlet, visit the edit page by clicking on the icon. This is a dynamic portlet that will refresh at 1 minute intervals.

Auto-Discovery

The Auto-Discovery portlet shows all new servers that have been discovered and need to be approved. Services are automatically approved into the HQ inventory without user intervention. By default, only the last 5 auto-discoveries will be shown. To modify this setting, click and proceed to "HQ Dashboard Settings: Auto-Discovery Queue" screen.

If the remove button is clicked, the selected servers will be permanantly removed from the queue. To import a server that has previously been removed, navigate to the Auto-Discovery Results page and select Import Server for the removed server. The Auto-Discovery Results page can be found by first clicking on the Inventory tab for the platform the server resides on. Then click on the Auto-Discovery sub-tab, then the View Results link.

Control Actions

The control actions portlet displays all recent control actions that have been performed on resources through the HQ portal. In addition to showing recent actions, upcoming scheduled actions, and the most frequent actions are also displayed.

Favorite Resources

The favorite resources portlet will show current usage and availability for your favorite resources. To add resources to this portlet, click and proceed to the "HQ Dashboard Settings: Favorite Resources" screen.

Metric Viewer

The metric viewer portlet(s) will show the specified metric values for selected resources. To change the specific metric or the resources shown, click . This is a dynamic portlet that will refresh at 1 minute intervals.

Problem Resources*

The problem resources portlet displays all resources with Problem Metrics.

Recently Added Resources

The recently added resources portlet shows all recent additions to the HQ inventory. By default only additions made in the past day are shown. To change this setting, click .

Recent Alerts

The recent alerts portlet(s) will show alerts for specified resources. To choose the resources to display alerts for, click and proceed to the "HQ Dashboard Settings: Alerts" screen. This is a dynamic portlet that will refresh at 1 minute intervals.

Recently Viewed Resources

The recently viewed resources portlet displays resources that you have viewed most recently in reverse chronological order.

Search Resources

The search resources portlet allows for searching of the HQ inventory.

Summary Counts

The summary counts portlet displays counts for each resource type that the user is allowed to view. By default only top-level resource counts are given. To display counts for each resource type, click and proceed to the "HQ Dashboard Settings: Summary Counts" screen. You can go directly to the Browse Resources section for the resource type by clicking on the the type name or count.
