Triggered Alerts

Available in Open Source

This screen allows users to view the alerts that have been triggered, organized by date, for the specific resource. For each triggered alert, this information is displayed:

  • Priority: Possible values = low, medium, high
  • Alert Date: The date and time the alert was triggered
  • Alert Definition: The name of the triggered alert
  • Alert Condition: The alert condition that was met that triggered the alert
  • Actual Value: If the alert condition is a metric comparison, this is the value that HQ measured for the metric (which then triggered alert).
  • Fixed: Whether or not a user has marked the alert as "fixed"
  • Ack: An icon in this column indicates that the alert has not yet been acknowledged. The lack of an icon indicates that the alert has been acknowledged.

Tasks Available on This Screen

On this screen, users can do the following tasks:

To mark an alert "fixed":

  • Check the alert(s) and click Fixed.

To look at alerts triggered on a different date:

  • Click or to navigate earlier or later than the current date.

To view the list of alerts for a specific date:

  • Click and select a specific date.

To acknowledge an alert:

  • Click on the alert's row in the "Ack" column.
    If the icon is not there, that means the alert has already been acknowledged. Acknowledgment of an alert can initiate an escalation scheme for the alert.

Navigating via the Masthead Menu

Main screens in Hyperic contain the same menu in the masthead. Users can navigate to the following parts of HQ using this menu:

Dashboard Takes users to the starting screen in HQ, which displays information about resource health, recent alerts, recently performed Auto-Discoveries, and control actions.
Browse Resources Takes users to the screen where they can locate and navigate through information about all managed resources
Administration Takes users (who are assigned to a role that has the Administer Hyperic HQ Server Configuration option enabled) to the screen where they can manage the Hyperic HQ Server configuration.
Recently Viewed Resources Provides a drop-down list of resources that the user has recently viewed. Selecting one of those resources takes the user to the inventory of that resource.
Help Opens screen-sensitive online help in a secondary window
