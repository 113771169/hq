HQ Dashboard Settings: Alerts

From this page you can edit the contents of the Recent Alerts portlet on the Dashboard.

To toggle the entire Recent Alerts portlet display on or off, select the YES or NO radio button.

Set the number of recent alerts to display from the drop-down list. Specify the priority of alerts you want displayed. This can be one of High, Medium, Low, or ALL. Alerts of priorities equal to or higher than your selection will be displayed. You can choose to display alerts from the past 30 minutes, hour, 12 hours, day, week, or month. Next, specify whether you want to display alerts for All Resources or a subset of Selected Resources. If you chose Selected Resources, check the boxes for the resources you wan to see alerts for in the Dashboard.

To add a resource to the list displayed on the Dashboard, click the ADD TO LIST button which will take you to the Add/Remove Resources page.
