Scheduling Control Actions

Available in Open Source.

This screen allows users to schedule a control action for a resource.

Availability of Control Actions

The control actions available on this screen depend on the resource type. Product-specific information about control actions can be found through the list of products managed by HQ.

Tasks Available on This Screen

On this screen, users can do one thing: schedule a control action.

To schedule a control action:

  1. Select a Control Action.
  2. (optional) Type a Description.
  3. Specify when the control action should run:
    1. Immediately (this equates to "on demand" or "quick start" on other control-action-related screens) or
    2. A date (month/dd/yyyy) and a time (hh:mm) and whether the action should Recur daily, weekly, or monthly (if at all)
