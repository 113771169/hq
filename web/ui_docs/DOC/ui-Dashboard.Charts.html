HQ Dashboard Settings: Charts

Available in HQ Open Source unless marked by * for HQ Enterprise only

This screen allows users to delete links to metric charts from the display in the "Saved Charts" portlet in the Dashboard. Charts can be added to the portlet only when viewing the specific chart.
