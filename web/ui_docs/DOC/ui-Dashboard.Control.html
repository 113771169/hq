HQ Dashboard Settings: Control Actions

Available in HQ Open Source unless marked by * for HQ Enterprise only

This screen allows users to change the display of control actions in the "Control Actions" portlet in the Dashboard.

If you have any comments or suggestions for this help page, please submit them at the bottom of the page by clicking Add Comment.

Tasks Available on This Screen

On this screen users can do one thing: change the display of control actions in the Dashboard's "Control Actions" portlet.

To change the display of control actions:

  1. Change the display of recently performed control actions:
    1. Check Last # completed in to display a specific number of completed control actions in a specific time range. (To stop the display of this data, uncheck it.)
    2. In the first drop-down list, select the number of completed control actions to display.
    3. In the second drop-down list, select the time range from which to gather that data.
  2. Change the display of control actions most frequently performed on-demand:
    1. Check Last # most frequently controlled to enable the display. (To stop the display of on-demand control-action data, uncheck it.)
    2. Select the number of the most frequently performed control actions to display.
