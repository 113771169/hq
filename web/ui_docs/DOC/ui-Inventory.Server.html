Viewing Server Inventory

Available in Open Source.

This inventory screen allows users to view and manage a selected server. The screen displays a server's user-specified properties, the services hosted by the server, groups containing the server, and the server's monitoring and control configuration. The server's name is at the rightmost of the path listed at the top of the screen.

Not all screen elements described in this wiki page are available for all instances of a server. Some server types simply have more or fewer data elements associated with them.

Tasks Available on This Screen

On this screen, users can do (or at least start) the following tasks:

To view a map representation of the server in relationship to its platform and hosted services:

  • Click in the upper left.
    The platform that hosts this server is displayed at the bottom of the map, and above it is displayed the server. Above the server are displayed all the services the server hosts. All resource names are bolded and the resource type is displayed just below in smaller type. From there you can navigate to the platform or any one of the hosted services by clicking its name.

To change the server's owner:

  • In the middle near the top of the screen, click Change next to the listed owner and proceed to the "Edit" screen.

To change the server's name and description:

  • In the "General Properties" section, click the top and proceed to the "Edit" screen.

To create a new service:

  • In the Tools Menu in the upper right, click New Service and proceed to the "Add a Service" screen.
    (If this server was auto-discovered, users cannot manually add this new or any service to it.)

To add this server to the user's Dashboard favorites:

  • In the Tools Menu in the upper right, select Add to Dashboard Favorites.

To change the install path of the server:

  • In the "General Properties" section, click the bottom and proceed to the "Edit" screen.

To add this server to a group:

  • In "Groups containing this resource," click and proceed to the "Add to Groups" screen.

To remove this server from a group:

  • In "Groups containing this resource," check the group and click .

To change the any of the server's configuration (log- and configuration-tracking, control actions, etc.):

  • In "Configuration Properties," click and proceed to the "Edit" screen.

To enable or disable auto-discovery of this server's services:

  • In "Configuration Properties," click and proceed to the "Edit" screen.

Sections on This Screen

Top Level

The top of the page displays high-level identifying information for the server being viewed. The data fields displayed here depend on the server type.

General Properties

This section lists the properties the user specified when creating or last modifying the server:

  • Description: A description of the server
  • Resource type: The server type
  • Date Created: The date the server was added to the HQ inventory
  • Date Modified, Modified By: The date the server properties were last modified and the user who modified them
  • Install Path: The path where this server is installed
  • Host Platform: The platform that hosts this server
  • Total Services: The total number of service instances auto-discovered on this server
  • Services By Type: The total number of distinct service types auto-discovered on the server (a categorization of the Total Services)

Services

This section lists the services discovered on this server. (For the most part, resources are auto-discovered, and therefore users cannot manually add to a list of such resources. Users can delete from the list, however, if they don't like HQ's auto-discovery results.) To view a service listed here, click the service's name. Each service's availability is also indicated.

Groups Containing This Resource

This section lists the groups of which this server is a member. To view a group listed here, click the group's name.

Configuration Properties

This section displays the shared, monitoring, and control configuration properties of the server.

HQ often provides reasonable defaults for the configuration properties, but HQ users must provide information specific to their environment.

Navigating via the Masthead Menu

Main screens in Hyperic contain the same menu in the masthead. Users can navigate to the following parts of HQ using this menu:

Dashboard Takes users to the starting screen in HQ, which displays information about resource health, recent alerts, recently performed Auto-Discoveries, and control actions.
Browse Resources Takes users to the screen where they can locate and navigate through information about all managed resources
Administration Takes users (who are assigned to a role that has the Administer Hyperic HQ Server Configuration option enabled) to the screen where they can manage the Hyperic HQ Server configuration.
Recently Viewed Resources Provides a drop-down list of resources that the user has recently viewed. Selecting one of those resources takes the user to the inventory of that resource.
Help Opens screen-sensitive online help in a secondary window
