Editing a Role's Properties and Permissions

Available only in HQ Enterprise

This screen allows users to edit the general properties and the permissions affiliated with a role. Users must have the Hyperic HQ Administrator privileges to do this. The dashboard associated with this role is a constant.

If you have any comments or suggestions for this help page, please submit them at the bottom of the page by clicking Add Comment.

Tasks Available on This Screen

On this screen, users can do the following tasks:

To change the role's general properties:

  • Change the Name or Description (optional) and click .

To change the role's owner:

  • Click Change next to Owner.

To change the permissions assignment of the role:

  1. Find the categories or inventory levels to which the role (and, by extension, the users assigned to the role) will have access and check the level of access the role will have.
  2. Click .
The Final Step

Even if users are assigned to the role, the role's (and users') permissions will not apply to any specific resources until a group (of said specific resources) is also assigned to the role.
