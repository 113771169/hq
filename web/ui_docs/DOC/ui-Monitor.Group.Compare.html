Group Compare Metrics

Available in HQ Open Source unless marked by * for HQ Enterprise only

This screen displays all available metric data, in a format conducive to comparison across resources, for selected group members. If a resource does not have data for a listed metric, the resource is simply not listed with the metric.

Tasks Available on This Screen

On this screen, users can do the following:

To chart a metric for the listed resources:

  • Click the metric name and proceed to the "Metric Chart" screen.

To refresh the displayed metric values:

  • Click next to Get Current Values at bottom-right.

To change the metric display range:

  • Follow the instructions for doing the same thing on the "Current Health" screen.

Sections on This Screen

This screen organizes the metrics by category, which are explained here. For each metric, for each resource being compared, the screen displays the:

  • LOW: The lowest observed metric value
  • AVG: The average of the observed metric values
  • PEAK: The highest observed metric value
  • LAST: The last observed metric value
