HQ Dashboard Settings: Auto-Discovery

Available in HQ Open Source unless marked by * for HQ Enterprise only

This screen allows users to change the display of auto-discovered platforms in the "Auto-Discovery" portlet on the Dashboard.

If you have any comments or suggestions for this help page, please submit them at the bottom of the page by clicking Add Comment.

Tasks Available on This Screen

On this screen users can do one thing: set the number of completed auto-discoveries displayed in the Dashboard's "Auto-Discovery" portlet.
