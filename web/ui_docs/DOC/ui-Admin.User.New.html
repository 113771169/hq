Adding a User

Available in Open Source.

This screen allows users to create a new user. Users must have the "create user" permission in order to do this.

Tasks Available on This Screen

On this screen, users can do one thing: create a new user.

To create a new user:

  1. Fill in the following fields:
    • Name
    • Username
    • Phone
    • Email
    • Department
    • Password (and Confirm it): Passwords must contain at least 6 case-sensitive characters and numbers, and no spaces or quotation marks.
    • SMS Address
  2. In Enable Login, specify whether the user can log in or not.
  3. Click OK & Assign User to Roles.
    On the ensuing "Edit User" screen, users can click OK to create the user without assigning it to any role.
