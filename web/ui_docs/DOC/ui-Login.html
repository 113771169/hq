Logging In

This screen allows users to log in to HQ. The Dashboard is the initial screen presented to users in HQ.

Tasks Available on This Screen

On this screen, users can do one thing: log in to HQ.

To log in to HQ:

  • Type a valid Login (username) and Password and click Sign In.
    HQ is installed with a default HQ Administrator username and password of hqadmin and hqadmin.
