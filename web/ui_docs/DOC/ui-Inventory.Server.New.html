Adding a Server to the HQ Inventory

This screen allows users to create a new server and add it to a platform. (Servers can be created only within the context of a platform.) Users might manually add a server (or service or platform) under the following circumstances:

  • It is a technology that is not monitored by default by HQ.
  • It is a custom plugin.
  • It is installed in a nonstandard location that HQ does not look in by default.
  • It has a nonstandard name in the process list.

Tasks Available on This Screen

On this screen users can do one thing: create a new server and add it to a platform.

To add a new server:

  1. Fill in the following fields:
    • Name: The name of the server
    • Description: (optional) A description of the server
    • Server Type: The server type
    • Install Path: The full path of the directory where the server software is installed
  2. Click .

The server's Owner defaults to the account that created the server. And the Host Platform is the name of the platform this server is being added to.
