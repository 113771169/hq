Schedule Maintenance  for a Group*

Available in HQ Open Source unless marked by * for HQ Enterprise only

This page allows you to specify a period of time during which metric collection for the resources in a group is paused. During the maintenance window, alerts that are active for the resources will not fire. The Schedule Maintenance page is available on the Tools menu when a group is selected.

If you have any comments or suggestions for this help page, please submit them at the bottom of the page by clicking Add Comment.

To schedule maintenance:

  1. Navigate to the group for which you which to schedule maintenance.
  2. Choose Schedule Maintenance from the Tools menu. 
  3. On the Schedule Downtime page, enter the start and end dates and times for the maintenance window and click Schedule.

You can reschedule the maintenance period by following the procedure described above, changing the maintenance window, and clicking Reschedule.
