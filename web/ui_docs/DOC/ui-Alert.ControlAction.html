Assigning a Control Action to an Alert Definition

Available in HQ Enterprise

This screen allows users to view and manage the existing control action defined on an alert definition and to create one if it doesn't already exist. A maximum of one control action is allowed per alert definition. The control action in an alert definition needn't be performed on the resource for which the alert is defined; an alert defined for resource A can include a control action to be performed on resource B.
