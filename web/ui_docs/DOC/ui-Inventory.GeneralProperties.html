Editing a Resource's Name and Description

Topics marked with * relate to HQ Enterprise-only features.

This screen allows users to edit a resource's general properties, namely its name and description.

Feedback is welcome. Click Add Comment at the bottom of the page.

Tasks Available on This Screen

On this screen, users can do one thing: edit a resource's name and description.
