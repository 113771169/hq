Browsing Resources

Available in HQ Open Source unless marked by * for HQ Enterprise only

The Resource Hub page, available when you click the Resources tab in the masthead, allows you to list managed resources in HQ, navigate to a resource's Monitor, Inventory, or Alerts page, and create Platforms, Groups, and Applications.

Tasks Available on This Page

To change or filter the displayed list of resources:

  1. To display all resources of a certain inventory type - Platform, Server, Service, Compatible Groups/Clusters, Mixed Groups, or Applications - click the desired link in the bar at the top of the resource list.
    Each link indicates the number of resources of that type.

To search for specific resources:

You can specify one or more search criteria: 

  • Type part or all of the resource's name in Search. The search is case-insensitive.
  • Filter the list by resource type by selecting a type from the pull-down list.
  • Limit the list to unavailable resources by clicking the Unavailable box.
  • Limit the list to resources owned by the user you are logged in as by checking the Owned by box.

Click the Any or All radio button to list resources that meet all of the criteria, or those that meet any of the criteria, respectively.

Click to run the search.

To view resources graphically:

  • Click in the upper right-hand corner.

To view resources textually:

  • Click in the upper right-hand corner.

To navigate to a resource's inventory, monitoring, or alert page:

  • Click the appropriate icon next to the resource name. (Simply clicking the resource name will display the resource's monitoring page.)
    If an particular icon is not shown for a resource, the associated page is not provided for that resource type. For instance, there is no Monitor page for a Mixed Group.
    Icon Action
    "Monitor" page
    Inventory page (for a platform, server, service, group, application)
    "Alert" page
    Indicates this feature is not available due to permissions.

To create a Platform:

  • Click New Platform in the Tools Menu and proceed to the "New Platform" page.
    Note:  you cannot create a platform of a type that HQ can auto-discover.

To create an Applications:

  • Click New Application in the Tools Menu and proceed to the "New Application" page.

To create a group:

  • To create a group out of listed resources, check the resources and click and proceed to the "New Group" screen.
    This group will already have the selected resources as members.
  • To create a group from scratch, click New Group in the Tools Menu and proceed to the "New Group" screen.
    This group will start with no member resources.

To delete a resource from the HQ inventory:

  • Check the resource and click .
    The resource is permanently removed from the HQ inventory, and therefore cannot be monitored or controlled.

Navigating via the Masthead Menu

Main screens in Hyperic contain a masthead menu. Users can navigate to the following parts of HQ using this menu:

Dashboard Takes users to the Dashboard, the starting screen in HQ, which displays information about resource health, recent alerts, recently performed auto-discovery scan, and control actions
Resources Click the menu name to access either Browse (takes users to the "Browse Resources" screen, where they can locate and navigate through information about all managed resources), Currently Down (takes users to the "Currently Down" screen, where they can look at all managed resources that are unavailable), Nagios Availability (available only when Nagios is installed), or Recently Viewed (provides a drop-down list of resources that the user has recently viewed. Selecting one of those resources takes the user to the "Current Health" screen for the resource).
Analyze Click the menu name to access either Reporting * (takes users to the "Reporting" screen, where they can generate all reports available in HQ), Alert Center (takes users to the Alert Center, where they can look at all alerts triggered across the system).
Administration Takes users (with "Administer Hyperic HQ Server Configuration" permissions) to the "Administration" screen, where they can manage the system-wide settings
Search box The Search box in the upper right of the Masthead allows you to search the HQ inventory for resources of any type.

Search results appear after four characters are entered in the text box. Results will include the first 10 platforms, servers, services, and groups whose name includes the search string.

Double-click a resource in the list to navigate to its Resource page.

The masthead also displays the two most recently triggered alerts. Users can click the alert's time to be taken to the alert detail.
