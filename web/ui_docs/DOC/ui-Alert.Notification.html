Edit Alert Add Notification

Available in Open Source

This screen allows users to select the HQ roles (and thereby all users with that role), specific HQ users, or email addresses (not necessarily of HQ users) that should be notified when an alert is triggered.

If you have any comments or suggestions for this help page, please submit them at the bottom of the page by clicking Add Comment.

Tasks Available on This Screen

On this screen users can do the following tasks:

To select roles or users to notify when the alert is triggered:

  1. Check the desired roles or users at the left.
  2. Click between the two lists to move the selected items to the right.
    The arrow is enabled only when some resources are selected.
