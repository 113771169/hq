HQ Dashboard Settings: Add Resources

Available in Open Source

This screen allows users to select which resources appear in one of several portlets on the Dashboard. (Which portlet is affected depends on how users get to this screen.)

Tasks Available on This Screen

On this screen users can do one thing: select a set of resources.

To select a set of resources:

  1. Check the desired resources in "Resources" and click between the two lists to move the selected resources to "Add Resources." (The arrow is enabled only when some resources are selected.)
    Users can filter the list of all available resources either by:
    • Typing in (part of) the resource's name in Filter By Name and clicking directly to the right, or
    • Selecting a resource type or inventory level in Filter By Type.
  2. To remove resources from the desired set, check the resources in "Add Resources" and click between the two lists to move the selected resources back to "Resources." (The arrow is enabled only when some resources are selected.)
