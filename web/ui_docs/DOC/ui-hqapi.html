HQ API

The HQ API is a web services API, built-in to HQ, that allows programmatic access to the HQ Server.  In Hyperic HQ 4.0, the HQ API replaced the Command Line Interface (CLI) available in previous versions of HQ. 

In the HQ user interface, click the HQ Web Services API link to download a package that contains:  

  • The hqapi1.jar client library
  • Support jars required to run the library
  • The XSD document that defines the web service.
  • The Javadoc documentation for the API.

The HQ API supports two methods of interaction:

  • Programmatic access through the Java API.
  • Script-based access using command line tools.

Documentation for the HQ API is available at HQ Web Services API.

The SVN repository has Test code.
