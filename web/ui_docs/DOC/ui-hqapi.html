HQ API 

The HQ API is a web services API, built-in to HQ, that allows programmatic access to the HQ Server.  In Hyperic HQ 4.0, the HQ API replaces the Command Line Interface (CLI) available in previous versions of HQ. 

In the HQ user interface, click the HQ Web Services API link to download a package that contains:  

  • The hqapi1.jar client library
  • Support jars required to run the library
  • The XSD document that defines the web service.
  • The Javadoc documentation for the API.

The HQ API supports two methods of interaction:

  • Script-based interaction with XML using curl. The SVN repository has many Shell Examples.
  • Programmatic access through the Java API. Java examples are planned, but have not been written yet. In the meantime, examples of using the API can be found in the Test Suite.

The SVN repository for the HQ API is http://svn.hyperic.org/projects/hqapi/trunk

To test and deploy into a running server, use the following instructions

svn co http://svn.hyperic.org/projects/hqapi/trunk hqapi
cd hqapi
ant deploy
ant test

Any failures to the test suite will be placed in build/testresults.
