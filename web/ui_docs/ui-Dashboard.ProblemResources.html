HQ Dashboard Settings: Problem Resources

From this page you can edit the contents of the Recent Alerts portlet on the Dashboard.

To toggle the entire Problem Resources portlet display on or off, select the YES or NO radio button.

Set the maximum number of problem resources to display from the Show Maximum of drop-down list. The For the Last drop-down allows you to specify the time range to take into account when evaluating problem resources. If there are problem resources, the entire list would be displayed on this page, regardless of the setting in the maximum drop-down.

To save your changes and return to the Dashboard , click OK.
