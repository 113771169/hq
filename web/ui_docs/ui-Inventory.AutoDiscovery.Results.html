Viewing Auto-Discovery Results

Available in Open Source.

This screen allows users to view the results of an auto-discovery scan on a platform.

  • Tasks available on this screen
  • Sections on this screen
  • Navigating via the top-level menu

Tasks Available on This Screen

On this screen, users can do the following tasks:

To view only changes to the platform since the last scan:

  • Select "New & Modified" in either of the two View fields.

To import information newly discovered since the last scan into the HQ Inventory:

  • Select "Import Values" or "Import Server" and click OK.

Sections on This Screen

Platform Type

This section lists high-level identifying information for the platform, including the platform's type, Fully Qualified Domain Name, and whether or not the auto-discovery scan discovered any changes in the platform's configuration since the last scan. Auto Discovery Detail shows "Modified Properties" when the scan did discover changes since the last scan, "Unchanged" when the scan didn't discover any changes (the platform is the same as it was during the last scan), ?

In Action, users can choose to import the discovered changes (if any) into the HQ Inventory.

Network Properties

This section lists identifying networking information for the platform. Users can look only at information that has changed since the last scan ("New & Modified" in View), only information that has not changed ("Unchanged" in View), or both ("All States"). Each platform has at least one set of IP Address, MAC Address, and Netmask — for the loopback (local) IP address, 127.0.0.1 — and then an additional set for each network interface discovered on the platform. Auto Discovery Detail shows "Modified Properties" when the scan discovered changes to these identifying data since the last scan and "?" when the scan didn't discover any changes. In Action, users can choose to import the discovered changes (if any) into the HQ Inventory.

Servers

This section lists servers discovered on the platform. As in the "Network Properties" section, users can look only at servers that were newly discovered in this scan or that had newly discovered other information ("New & Modified" in View), only servers that were already known about ("Unchanged" in View), or both ("All States"). In Action, users can choose to import the newly discovered servers (if any) into the HQ Inventory. This section lists the following information about each server:

  • Server: The server's name
  • Server Type: The server type
  • Install Path: The location on the platform where the server is installed
  • Server Status: Whether or not HQ already knew about the server ("unchanged") or didn't ("modified")
