Viewing All Users

Available in Open Source.

This screen allows users to view and manage all HQ users.

  • Tasks available on this screen
  • Navigating via the top-level menu

Tasks Available on This Screen

On this screen, users can do (or at least start) the following tasks:

To create a new user:

  • Click New at the bottom left or New User in the top panel and proceed to the "New User" screen.

To delete a user from the HQ system:

  • Check a user and click Delete.

To email a user:

  • Click the user's email address.

To view or edit a specific user:

  • Click the user's UserName and proceed to the "View User" screen.

To view and manage user roles:

  • Click List Roles in the top panel and proceed to the "List Roles" screen.

To create a new user role:

  • Click New Role in the top panel and proceed to the "New Role" screen.

Navigating via the Top-Level Menu

Main screens in Hyperic contain the same menu at the top of the page. Users can navigate to the following parts of HQ using this menu:

Dashboard Takes users to the starting screen in HQ, which displays information about resource health, recent alerts, recently performed Auto-Discoveries, and control actions.
Browse Resources Takes users to the screen where they can locate and navigate through information about all managed resources
Administration Takes users (who are assigned to a role that has the Administer Hyperic HQ Server Configuration option enabled) to the screen where they can manage the Hyperic HQ Server configuration.
Recently Viewed Resources Provides a drop-down list of resources that the user has recently viewed. Selecting one of those resources takes the user to the inventory of that resource.
Help Opens screen-sensitive online help in a secondary window
