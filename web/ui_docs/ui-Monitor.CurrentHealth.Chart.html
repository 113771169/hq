HQ Chart

This page displays the metric data for the selected metric(s). Each column represents a 1/60 time slice of the current metric display range. Each data column is displayed with the range values collected in that time slice (the I bar), as well as the actual peak, average and low values and, if available, the baseline, high range and low range values.
