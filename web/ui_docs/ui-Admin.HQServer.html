HQ Edit Server Settings

This page contains the general configuration settings for the HQ server.

HQ Email Configuration Properties

The email configurations apply to the alert emails sent by HQ. The base URL is used as the prefix to the URL links to the alert details in the email notification. The URL should be the base URL to the HQ server. From Email Address is used as the sender of the alert emails.

Data Manager Configuration Properties

The Data Manager Configuration Properties affects the performance of HQ by dictating how the data is condensed and purged out of the database. While metric data is maintained for up to two years, modifying how quickly detailed data and alerts get purged can improve performance dramatically.

Automatic Baseline Configuration Properties

By default, Hyperic HQ automatically calculates the baseline values for all dynamic metrics. You can affect the accuracy of the automatically calculated baselines by determining how frequently the baseline is recalculated, how large a dataset to use, and the minimum number of data points required to calculate a new baseline.

LDAP Configuration Properties

If you would like to use LDAP authentication as your backend, select the Use LDAP Authentication checkbox and fill in the necessary configuration settings.
