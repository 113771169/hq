Adding a Role

Available in Enterprise Edition

This screen allows users to create a new role. Users must have the Hyperic HQ Administrator priveleges to do this.

Tasks Available on This Screen

On this screen, users can do one thing: create a new role.

To create a new user:

  1. Fill in the following fields:
    • Name
    • Description (optional)
  2. Find the categories or inventory levels to which the role (and, by extension, the users assigned to the role) will have access and check the level of access the role will have.
  3. Click .
    On the ensuing "View Role" screen, users can assign users and groups to the role.
The Final Step

Even if users are assigned to the role, the role's (and users') permissions will not apply to any specific resources until a group (of said specific resources) is also assigned to the role.

The role's Owner defaults to the account that created the role.
