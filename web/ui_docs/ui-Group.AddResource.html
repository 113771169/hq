Adding Resources to a Group

Available in Open Source

This screen allows users to add a resource to a group. If users arrive at this screen from viewing a group, they can add or remove multiple resources from that one group. If users arrive at this screen from viewing a platform, server, or service, they can add that one resource to multiple groups.

Tasks Available on This Screen

On this screen, users can do one the following tasks:

From a Group's Perspective

The only resources available for addition to the group are those of the same resource type specified for this group.

To add resources to a group:

  1. Check the desired resources in "Resources" and click the right arrow button between the two lists to move the selected resources to "Add Resources." (The arrow is enabled only when some resources are selected.)
    Users can filter the list of all available resources either by:
    • Typing in (part of) the resource's name in Filter By Name and clicking the arrow button directly to the right, or
    • Selecting a resource or inventory type in Filter By Type. (This filter is available only for mixed groups, as compatible groups contain only one resource type.)
  2. Click OK.

To remove resources from a group:

  1. Check the desired resources in "Add Resources" and click the left arrow button between the two lists to move the selected resources to "Resources." (The arrow is enabled only when some resources are selected.)
  2. Click OK.

From a Resource's Perspective

The only groups available for selection are compatible groups of the exact type of the resource in question and mixed groups. (In the case of an application, only mixed groups of type "Application" are available.) Groups that the resource is already a member of are not listed for selection.

To add the resource to groups:

  1. Check the desired groups in "Groups containing this resource" and click the right arrow button between the two lists to move the selected resources to "Add to Groups." (The arrow is enabled only when some resources are selected.)
  2. Click OK.
