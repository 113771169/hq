New Alert Definition

This page allows you to create new alert definitions for a specific resource in HQ. There are several required fields indicated by the asterisk, however many of them are populated by default. Generally, all that is necessary for a new alert definition is:

  1. A name for the alert
  2. One or more conditions to alert on
