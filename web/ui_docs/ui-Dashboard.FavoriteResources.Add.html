HQ Dashboard Settings: Add/Remove Resources

From this page you can select which resources should appear on on the Dashboard.

  • From the drop-down list, select the resource type of the resource you want to add.
  • The resources of the type you selected are displayed in the Resources list.
  • Select the resource you want to add to the Dashboard display by checking the box next to its name.
  • The transfer arrow becomes active.
  • Click the transfer arrow to add the resource to the Add Resources list.
  • Click OK to finialize the changes.
