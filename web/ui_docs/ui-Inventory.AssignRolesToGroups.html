Assigning Roles to Groups

Available in Open Source

This screen allows users to assign existing roles to an existing group, or vice-versa, depending on whether users are working within a group (then they will add a role to that group) or with a role (then they will add a group to that role). It's all the same in the end.

Tasks Available on This Screen

On this screen, users can do one thing: assign roles to a group.

To assign a role to a group:

  • Check the desired roles in "Roles" and click the right arrow button between the two lists to move the selected roles to "Add To Roles." (The arrow is enabled only when some roles are selected.)
