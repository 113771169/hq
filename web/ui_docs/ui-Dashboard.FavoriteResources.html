HQ Dashboard Settings: Favorite Resources

From this page you can edit the contents of the Favorite Resources portlet on the Dashboard. To toggle the entire Favorite Resources portlet display on or off, select the YES or NO radio button. Choose what resource health attributes you want displayed: Availability , Performance, Usage, Utilization.

To add a resource to the list displayed on the Dashboard, click the ADD TO LIST button at the bottom of the List Resources section. This link will take you to the Favorite Resources Add/Remove Resources page.

To rearrange the order of the resources, simply drag-and-drop individual resources to the order desired.
