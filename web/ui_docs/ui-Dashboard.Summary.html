HQ Dashboard Settings: Summary Counts

From this page you can customize the summary count information displayed on the Dashboard.

To toggle the entire Summary Counts portlet display on or off, select the YES or NO radio button.

Choose what Summary Counts you want displayed: Applications, Services, Platforms, Clusters, Servers, Groups. To save your changes and return to the Dashboard, click OK.
