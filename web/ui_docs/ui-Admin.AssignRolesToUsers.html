Assigning Roles to Users

Available in Enterprise Edition

This screen allows users to assign existing roles to an existing user, or vice-versa, depending on whether users are working within a user account (then they will add a role to that user) or within a role (then they will add a user account to that role). It's all the same in the end.

Tasks Available on This Screen

On this screen, users can do one thing: assign roles to a user.

To assign a role to a user:

  1. Check the desired roles in "Roles" and click between the two lists to move the selected roles to "Add To Roles." (The arrow is enabled only when some roles are selected.)
