Changing a Resource's Owner

Available in Open Source.

This screen allows users to change the owner of a single resource. In order to change a resource's owner, you must either have Administrator access or be the resource owner.

Tasks Available on This Screen

On this screen, users can do one thing: change a resource's owner.

To change a resource's owner:

  1. Click next to the new owner's name.
